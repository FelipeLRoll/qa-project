|-------|-------|
| **ID** | TC011 |
| **Título** | Aplicar filtro de marca + preço simultaneamente |

| **Prioridade** | 🟡 Média |
| **Tipo** | Funcional - Positivo |

**Objetivo:**

Verificar se múltiplos tipos de filtros funcionam em conjunto.



**Pré-condições:**

- Página inicial carregada



**Dados de Teste:**

```

Marca: Samsung

Preço: $500-$1000 (ajustar conforme disponível)

```



**Passos para Execução:**

1. Selecionar filtro "Samsung"

2. Selecionar filtro de preço

3. Verificar produtos resultantes



**Resultado Esperado:**

✅ Apenas produtos Samsung NA faixa de preço são exibidos

✅ Ambos os filtros estão ativos

✅ Lógica AND aplicada corretamente

✅ Contador correto
